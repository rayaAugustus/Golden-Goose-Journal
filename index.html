Golden Goose Journal

“理财先理心，财富是自信和行动力的副产品。” 这是面向理财初学者的记账 + 成长日记产品，帮助用户形成“记录—反思—行动”的闭环习惯。

记账 成长日记 目标管理 可视化洞察 复盘报告

产品目标与用户画像

  • 产品目标：为初学理财者提供记账 + 成长日记的一体化体验。
  • 核心用户：18-35 岁、理财认知初级的学生/职场新人/自由职业者。
  • 用户痛点：记账门槛高、日记难坚持、目标缺乏反馈。

MVP 功能清单

P0 记账 记录收入与支出，支持分类、账户、备注、时间。
P0 日记 记录心情与反思，支持标签与关键词检索。
P0 目标管理 设定理财/成长目标，拆解里程碑与提醒。
P1 可视化 通过图表展示收支结构、趋势与目标进度。
P1 复盘报告 按周/月生成财务 + 成长复盘摘要。
P2 习惯打卡 轻量打卡强化“记录-行动”闭环。

关键用户流程

1. 记账（添加一笔支出）

  • 入口：底部导航“记账” → “+ 新增”。
  • 字段：金额、类型、分类、账户、日期时间、备注、标签。
  • 交互：金额输入后展示本月同类平均提示。

2. 日记（写一篇成长日记）

  • 入口：底部导航“日记” → “+ 新建”。
  • 字段：标题、正文、心情、标签、关联账目。
  • 交互：支持快速模板与目标管理联动。

3. 目标管理（创建目标）

  • 入口：底部导航“目标”。
  • 字段：目标名称、类型、目标值、截止日期、提醒频率。
  • 交互：模板化目标与里程碑完成提示。

4. 可视化（查看月度收支）

  • 入口：底部导航“洞察”。
  • 页面：趋势折线、分类占比、目标完成度。
  • 交互：点击分类查看明细与时间维度切换。

验收标准

  • 记账：新增/编辑/删除账目，支持过滤与数据持久化。
  • 日记：新增/编辑/删除日记，支持标签与关键词检索。
  • 目标管理：可创建/编辑/关闭目标，展示进度。
  • 可视化：展示趋势与分类占比，支持时间维度切换。
