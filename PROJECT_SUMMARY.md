## 🎯 核心用户流程

### 每日主流程
```
打开App → 今日页 → 写日记(3条成功) → 完成 
    ↓
提示创建挑战 → 填写标题和最小动作 → 创建成功
    ↓
挑战倒计时开始 → 收到提醒 → 完成挑战
    ↓
提示存入梦想罐 → 选择金额 → 存入成功
```

### 低谷救援流程
```
感到低落 → 打开日记页 → 点击"成功回顾"
    ↓
随机展示历史成功 → 重拾信心
    ↓
提示创建新挑战 → 重新开始行动
```
